Merge_Small_File_to_HDFS
========================

--------------------------------------------
This util can merge lots of small files to hdfs.<br>
Benchmark: <br>
  37GB, 75,000 files<br>
  costs about 20mins.

BTW:
This program faster than Hadoop Put method(One 37GB file).
